Privacy Policy for Notes Sidebar

Notes Sidebar is a browser extension that lets users save selected text from webpages into a side panel for note-taking and organization.

Data the extension processes
- Selected text that the user explicitly chooses to capture
- Related page information such as page title and page URL when needed to organize saved notes by page
- Locally stored extension data such as merged notes, page groups, and user-edited note content

How data is used
- To save, display, organize, edit, copy, and manage notes inside the extension
- To support page-based note grouping and merged note views
- To provide the extension’s core functionality only

How data is stored
- Data is stored locally in the user’s browser using Chrome extension storage

Data sharing
- Notes Sidebar does not sell user data
- Notes Sidebar does not use user data for advertising
- Notes Sidebar does not transfer user data to third parties except as required for the extension’s core functionality or by law

Remote code
- Notes Sidebar does not load or execute remote code
